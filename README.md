# sm_loadingscreen
A clean and simple loading screen for FiveM servers. Easy to install, lightweight, and fully customizable.


## Installation
1. Download the script.
2. Place the folder into your `resources` directory.
3. Add `ensure sm_loadingscreen` to your `server.cfg` file.

## Customization
You can modify the HTML, CSS, and JS files included in the folder to change the background image, server name, or music to match your server's style.
